# 迷你農場 🌾

一個可在**電腦瀏覽器**和**手機瀏覽器**上玩的 2.5D 等距農場經營遊戲。
你扮演一位農夫：**種田、蓋畜舍、養牛羊雞、捏臉、進城買衣服換裝**，進度**自動存檔**。
純 HTML/CSS/JavaScript，**免安裝、免打包**。

> 🌐 線上遊玩：https://a26222239-sketch.github.io/Simulation-Games/

---

## 操作

| 動作 | 電腦 | 手機 |
|------|------|------|
| 移動 | WASD / 方向鍵 | 左下虛擬搖桿 |
| 互動 | 點地圖格子 | 點地圖格子 |

底部工具列：**⛏️鋤地 → 🌱種植 → 💧澆水 → ✋收成/互動 → 🏗️建造/畜牧**。
右側按鈕：**💤睡覺(下一天)、🏘️進城/回農場、👤捏臉、👕衣櫃、☰選單(存讀檔)**。

### 遊戲循環（第一版「能玩薄片」，6 大系統都有）
1. **種田**：⛏️鋤草地成農地 → 🌱種子 → 💧澆水 →（睡覺換日）作物長一階 → 長成後 ✋收成進庫存。
   *只有澆過水的作物隔天才會長大。*
2. **蓋畜舍**：🏗️ 選雞舍/畜舍 → 點農場空地放置。
3. **養動物**：🏗️ 買雞/牛/羊（需先有對應畜舍）→ 每天換日產出蛋/奶/毛 → 走到動物旁 ✋撿取。
4. **賣錢**：走到農場的「出貨箱」✋ → 一次賣出所有庫存。
5. **捏臉**：👤 調膚色/髮型/髮色/服裝（正面立繪預覽）。
6. **進城換裝**：🏘️進城 → 走到「服飾店」點一下買衣服、走到「種子店」選作物；👕衣櫃隨時換裝。

### 本機執行
```bash
python3 -m http.server 8000   # 開 http://localhost:8000（用了 ES Modules，需透過伺服器）
```

## 換成你的 GPT 立繪 🎨

目前角色與物件都是程式畫的。之後要換成你的立繪：

- **角色換裝（正面立繪）**：每件衣服一張正面 PNG、髮型分層 PNG，改 `js/character.js` 的
  `drawPortrait` 改成 `drawImage` 疊圖即可（捏臉/衣櫃會直接顯示）。
- **世界物件**：作物、畜舍、動物、地磚可在 `render.js` 加 `getSprite` 載入
  `assets/*.png` 後改用 `drawImage`（其他遊戲版本已有此機制可參考）。

各項外型建議寫在 `js/config.js`（`ART_NOTE` 與各資料的說明）。

## 引擎：Phaser 3

本作改用 **Phaser 3** 遊戲引擎驅動（場景、相機跟隨、輸入、主迴圈、畫面縮放），
引擎本體放在 `vendor/phaser.min.js`（本地檔，不依賴外部 CDN）。
**遊戲邏輯（`world.js`/`config.js`/`player.js`）與引擎無關**，所以日後要換引擎或加功能都好處理。

目前畫面用 Phaser 的 `Graphics` 即時繪製（`draw.js`）。等你有了 GPT 立繪/精靈圖，
可逐項改成 `this.load.spritesheet` + `this.add.sprite`（搭配 `this.anims` 走路動畫），
深度遮擋用 `sprite.setDepth(gx+gy)`。

## 檔案結構

```
index.html          畫面與 UI（HUD、搖桿、工具列）
css/style.css       樣式（含手機響應式、彈窗、搖桿）
vendor/phaser.min.js Phaser 3 引擎本體
js/config.js        規則資料：地圖、作物、動物、畜舍、衣服、外觀選項
js/iso.js           等距座標換算
js/player.js        主角：位置、面向、移動/尋路跟隨、外觀
js/world.js         世界狀態：農場/城鎮、作物、畜舍、動物、買賣、換日、BFS尋路、存讀檔
js/character.js     角色正面立繪（捏臉/衣櫃預覽，DOM canvas）
js/draw.js          Phaser Graphics 繪製（地面/作物/建築/動物/主角，深度排序）
js/ui.js            彈出介面：捏臉/衣櫃/服飾店/種子店/建造畜牧
js/phaser-main.js   進入點：Phaser 場景、相機、輸入、HUD、組裝啟動
assets/             放你的立繪
```

> 備註：採場景(scene)概念（農場/城鎮），日後擴充（更多作物、季節、釣魚、多城鎮等）都好接。
> 先前的「城市建造」「商店經營」與「純 Canvas 農場」版本都保留在 git 歷史中。
