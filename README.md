# 迷你模擬城市 🏙️

一個可以在**電腦瀏覽器**和**手機瀏覽器**上玩的 2.5D 等距城市建造遊戲。
可以蓋道路、住宅、商店、公園，經營人口與資金，並且**自動存檔**。

純 HTML / CSS / JavaScript 寫成，**不需要安裝任何東西、也不需要打包**。

---

## 怎麼玩（本機）

把整個資料夾下載下來，用瀏覽器打開 `index.html` 就能玩。

> ⚠️ 注意：因為程式用了 ES Modules（`import`），少數瀏覽器直接用
> `file://` 開啟會擋住模組載入。如果打開後一片空白，請改用本機伺服器：
>
> ```bash
> # 在專案資料夾裡執行（擇一）
> python3 -m http.server 8000
> # 然後瀏覽器開 http://localhost:8000
> ```

### 操作方式

| 動作 | 電腦 | 手機 |
|------|------|------|
| 蓋 / 拆建築 | 點一下格子 | 點一下格子 |
| 平移地圖 | 按住拖曳空白處 | 單指拖曳 |
| 縮放 | 滑鼠滾輪 | 雙指撥動 |

下方工具列選要蓋什麼，右上 ☰ 是選單（存檔 / 讀檔 / 匯出 / 匯入 / 重新開始）。

### 遊戲機制

- **道路**是基礎：住宅和商店必須**鄰接道路**才會運作。
- **住宅**接到路後人口會慢慢成長，居民繳稅 → 帶來收入。
- **商店**接到路後每月固定營收（有少量維護費）。
- **公園**目前是裝飾（有維護費），可作為之後「幸福度／地價」的擴充點。
- 上方資訊列即時顯示資金、人口、每月淨收入、月份。

## 存檔說明 💾

- **自動存檔**：每 10 秒、以及切到背景／關閉分頁時都會存。
- **手動存檔 / 讀檔**：右上選單。
- **匯出存檔**：下載成 `.json` 檔，可備份或換裝置。
- **匯入存檔**：把 `.json` 讀回來。

存檔放在瀏覽器的 `localStorage`（同一台裝置、同一個瀏覽器才看得到）。
換手機或換瀏覽器時，請用「匯出 / 匯入」搬移進度。

## 之後怎麼換上你用 GPT 生成的立繪 🎨

目前建築是用程式畫的「立體方塊」當佔位圖。等你用 GPT 生成好立繪後：

1. 把圖片（建議透明背景 PNG）放進 `assets/` 資料夾，例如
   `assets/house.png`、`assets/shop.png`。
2. 打開 `js/render.js`，找到 `drawBuilding()` 函式（裡面有 ★ 標記的註解）。
3. 在檔案最上方先載入圖片：
   ```js
   const sprites = {
     house: Object.assign(new Image(), { src: "assets/house.png" }),
     shop:  Object.assign(new Image(), { src: "assets/shop.png" }),
   };
   ```
4. 把 `drawBuilding()` 改成用圖片繪製（保留道路那段的平面畫法）：
   ```js
   const img = sprites[t.type];
   if (img && img.complete) {
     const w = TILE_W * z;
     const h = img.height * (w / img.width); // 維持原圖比例
     // 讓圖片底部對齊菱形中心
     ctx.drawImage(img, cx - w / 2, cy + (TILE_H / 2) * z - h, w, h);
     return;
   }
   ```

立繪的尺寸建議畫成「底部寬度約等於一格菱形寬」，這樣擺上去比例才對。

## 想加新的建築？

打開 `js/config.js`，在 `BUILDINGS` 裡照現有格式新增一筆，再到
`index.html` 的工具列加一顆按鈕（`data-tool="你的key"`）即可。
所有價格、收入、地圖大小、結算速度都集中在 `config.js`，方便調整。

## 檔案結構

```
index.html        畫面結構與 UI
css/style.css     版面與樣式（含手機響應式）
js/config.js      遊戲規則設定（建築、價格、地圖大小…）
js/iso.js         等距座標換算（格子 <-> 螢幕）
js/game.js        遊戲狀態與經濟邏輯、存檔/讀檔
js/render.js      把城市畫到畫布上
js/input.js       滑鼠 / 觸控操作
js/main.js        進入點：組裝起來並啟動
assets/           放你的立繪圖片
```
