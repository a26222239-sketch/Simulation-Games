# 迷你動物園 🦁

一個可在**電腦瀏覽器**和**手機瀏覽器**上玩的 2.5D 等距**動物園經營**遊戲（上帝視角）。
蓋步道、獸欄（養會走動的動物）、樹木、咖啡廳、紀念品店；遊客自己進場逛、看動物、消費，
靠**門票 + 商店營收**賺錢。用 **Phaser 3** 引擎，進度**自動存檔**。

> 🌐 線上遊玩：https://a26222239-sketch.github.io/Simulation-Games/

---

## 怎麼玩

下方工具列選工具，**點地圖放置**：
- 🟫 **步道**：遊客只走步道！要把步道從入口連到獸欄與商店，遊客才到得了。
- 🦁 **獸欄**：選一種動物 → 點空地蓋（含 2 隻動物，會在欄內走動）。**再點既有獸欄可加一隻動物。**
- ☕ **咖啡廳** / 🎁 **紀念品店**：遊客會去消費賺錢。
- 🌳 **樹木**：裝飾，增加「魅力」。
- 🚧 **拆除**：移除設施或步道。

**操作**：拖曳空白處平移畫面、滾輪 / 雙指縮放。右上 ☰ 可存讀檔 / 匯出匯入 / 重新開始。

**經營重點**：
- 上方顯示 資金 / 場內遊客 / 天數 / **魅力**。魅力越高（動物多、人氣高、有樹），遊客來得越快。
- 遊客進場付門票、到商店消費 → 收入；每過一天會扣動物與商店的維護費 → 別擴張太快。
- 一開始已幫你放好一個獅子獸欄 + 咖啡廳 + 步道，馬上就有遊客。

### 本機執行
```bash
python3 -m http.server 8000   # 開 http://localhost:8000
```

## 動物的 2.5D 動圖

動物在獸欄內走動，目前用程式繪製的動畫（身體、頭、腿擺動、依種類有鬃毛/長頸/象鼻等特徵）。
要換成你的 GPT 立繪：放精靈圖 `assets/animal_<id>.png`（建議 4 方向 × 4 走路格、透明背景），
在 `js/draw.js` 把 `animal()` 改成 `this.add.sprite` + `this.anims`（深度用 `setDepth(fx+fy)`）。
遊客、商店、樹木同理可換圖。

## 引擎與架構

- **Phaser 3**（`vendor/phaser.min.js`，本地檔不依賴 CDN）負責場景、相機（平移/縮放）、輸入、主迴圈。
- **遊戲邏輯與引擎分離**：`zoo.js` 是純邏輯，`draw.js` 只負責畫。

```
index.html          畫面與 UI（HUD、工具列、選單）
css/style.css       樣式（手機響應式、彈窗）
vendor/phaser.min.js Phaser 3 引擎
js/config.js        資料：地圖、動物、設施、遊客參數、價格
js/iso.js           等距座標換算
js/zoo.js           動物園狀態：放置、動物走動、遊客 AI(尋路/消費)、經濟、存讀檔
js/draw.js          Phaser Graphics 繪製（地面/獸欄/動物/商店/樹/遊客，深度排序）
js/ui.js            選動物種類的彈窗
js/phaser-main.js   進入點：Phaser 場景、上帝視角相機、放置、HUD
assets/             放你的立繪 / 動物精靈圖
```

> 先前做過的「城市建造」「商店經營」「農場（純 Canvas / Phaser）」版本都保留在 git 歷史中。
